## ✨ معرفی پروژه

**سینماچی** یک سیستم توصیه‌گر فیلم حرفه‌ای است که با استفاده از FastAPI و Frontend مدرن طراحی شده است. این پروژه شامل ۳۰ فیلم برتر جهان با اطلاعات کامل (نام، ژانر، امتیاز) می‌باشد.

## 🚀 امکانات پروژه

| شماره | قابلیت | توضیح |
|-------|--------|-------|
| 1 | 🎬 **۳۰ فیلم برتر** | لیست کامل ۳۰ فیلم معروف جهان |
| 2 | 🔍 **جستجوی پیشرفته** | جستجو بر اساس نام، ژانر و حداقل امتیاز |
| 3 | 🎯 **پیشنهاد فیلم** | دریافت فیلم‌های پیشنهادی بر اساس ژانر |
| 4 | 🏆 **فیلم‌های برتر** | نمایش N فیلم برتر با بالاترین امتیاز |
| 5 | 📊 **آمار کامل** | نمایش تعداد، میانگین امتیاز و توزیع ژانرها |
| 6 | 💾 **لیست من** | افزودن فیلم‌های مورد علاقه به لیست شخصی |
| 7 | 🔐 **سیستم عضویت** | ثبت‌نام و ورود با اعتبارسنجی قوی رمز عبور |
| 8 | 🎨 **طراحی مدرن** | تم شیشه‌ای (Glassmorphism) با انیمیشن‌های حرفه‌ای |
| 9 | 📱 **ریسپانسیو** | سازگار با موبایل، تبلت و دسکتاپ |

## 🛠️ تکنولوژی‌های استفاده شده

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** FastAPI
- **Database:** SQLite
- **Deployment:** Render
- **Icons:** Emoji & Font Awesome
- **Font:** Vazir (فارسی)
