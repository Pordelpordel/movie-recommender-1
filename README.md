## ✨ معرفی پروژه

**سینماچی** یک سیستم توصیه‌گر فیلم حرفه‌ای است که با استفاده از FastAPI و Frontend مدرن طراحی شده است. این پروژه شامل ۳۰ فیلم برتر جهان با اطلاعات کامل (نام، ژانر، امتیاز) می‌باشد و امکان جستجوی پیشرفته، دریافت پیشنهادات هوشمند و مدیریت لیست علاقه‌مندی‌ها را فراهم می‌کند.

## 🛠️ تکنولوژی‌های استفاده شده

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** FastAPI
- **Database:** SQLite
- **Deployment:** Render
- **Icons:** Emoji & Font Awesome
- **Font:** Vazir (فارسی)
